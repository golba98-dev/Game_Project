# Bigfoot Adventure

A 2D side-scrolling platformer game built with JavaScript and the [p5.js](https://p5js.org/) library. Guide the character through a procedurally generated landscape, collect coins, and avoid falling into canyons!

## üéÆ How to Play

Open `index.html` in your web browser to start the game.

### **Controls**
| Action | Key(s) |
| :--- | :--- |
| **Move Left** | `Left Arrow` or `A` |
| **Move Right** | `Right Arrow` or `D` |
| **Jump** | `Spacebar` or `W` |
| **Respawn** | `Spacebar` (when Game Over) |

### **Objective**
* Navigate the world by running and jumping.
* Collect coins scattered throughout the level.
* Avoid falling into canyons.
* If you fall off the screen, it's **GAME OVER**! Press Space to respawn and try again.

## ‚ú® Features

* **Side-Scrolling Camera:** The view follows the player as they explore the world.
* **Procedural Generation:** Mountains and trees are randomly generated every time you play, creating a unique landscape.
* **Physics System:** Includes gravity, jumping mechanics, and collision detection with the ground and canyons.
* **Parallax Backgrounds:** Clouds and scenery move at different rates to create depth.
* **Animated Character:** Custom drawing code for the character's movement and jump states.

## üìÇ Project Structure

* `index.html`: The main entry point that loads the game and libraries.
* `sketch.js`: Contains the main game logic, drawing functions, and physics engine.
* `p5.js`: The p5.js library file required for rendering and input handling.

## üöÄ Running the Game

1.  Download or clone the repository.
2.  Ensure `index.html`, `sketch.js`, and `p5.js` are in the same folder.
3.  Double-click `index.html` to launch the game in your default browser.
